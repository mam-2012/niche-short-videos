Privacy Policy

Last updated: 2026-04-07

Niche Short Videos Publisher ("the Tool") is operated by a single individual ("the Operator") for personal use only. This policy explains what data the Tool handles.

1. Data we collect

  • OAuth tokens issued by TikTok and YouTube to the Operator's own accounts. These are used solely to publish videos to those same accounts.
  • Video files and associated metadata (titles, descriptions, hashtags) generated by the Operator's own pipeline.

The Tool does not collect, store, or process any personal data of any other user.

2. How we store data

Tokens are stored locally on the Operator's machine in a configuration file with restricted file-system permissions. They are never transmitted to any third party other than the official API endpoints of the platforms they authorize (TikTok and Google).

3. Data sharing

No data is shared with anyone. The Tool communicates only with the official APIs of TikTok, Google (YouTube), Cloudflare R2, and Discord — each acting on behalf of accounts owned by the Operator.

4. Data retention and deletion

Tokens remain stored locally until the Operator either deletes the configuration file or revokes the Tool's access at:

  • TikTok: https://www.tiktok.com/setting/apps-and-website
  • Google: https://myaccount.google.com/permissions

5. Security

The Tool runs only on the Operator's own machine. It does not expose any public network service. OAuth flows use the official endpoints of each provider over HTTPS.

6. Changes

This policy may be updated from time to time. The most recent version is always available on this page.
