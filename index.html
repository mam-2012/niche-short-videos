Discord TikTok Publisher

Discord TikTok Publisher is an application that lets any user automatically publish short-form vertical videos to their own TikTok account after approving generated videos directly from Discord. Each user connects their own TikTok account and reviews content in their Discord server before anything is posted.

How it works

  1. The user signs in and connects their TikTok account via the official TikTok OAuth flow.
  2. The application generates short vertical videos for the user.
  3. Each generated video is delivered to the user's Discord channel for review.
  4. The user approves or rejects the video from Discord.
  5. On approval, the application uploads the video to the user's own TikTok account using the official TikTok Content Posting API.
